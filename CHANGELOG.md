# Changelog

本文件记录 ScriptFlow 的主要版本变化。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循项目当前发布记录。

## [Unreleased]

### Changed

- 重写 README，采用更常见的社区项目文档结构。
- 清理未使用的预设数据、旧平台元数据、孤立组件和未使用开发依赖。
- 默认首次打开改为空白工作区，不再加载内置示例预设。

## [0.1.9.3] - 2026-06-17

### Fixed

- 修复 Tiptap 编辑器 `onDocumentChange` 触发的无限更新循环。
- 修复目录和段落元信息重复 `setState` 导致的额外渲染。
- 增加旧版 React Flow edge handle 兼容迁移，避免旧工作区或模板因无效 handle 持续报警。

### Changed

- 媒体资源、模板与工作区数据统一向 IndexedDB 存储收敛。
- 保存工作区时保留自定义连接点数据，避免旧边引用丢失。

## [0.1.9.2] - 2026-06-17

### Added

- 支持在节点边缘通过快捷操作添加额外连接点。
- 新增图片节点显示偏好，可设置新建图片节点默认使用纯图或图文模式。
- 图片节点支持纯图片模式，减少描述区域对图片展示的占用。

### Changed

- 优化节点连接点交互和光标提示。
- 优化节点模板面板和底部工具栏的视觉表现。
- 调整部分节点默认内容，减少新建卡片后的占位文字干扰。

## [0.1.9.1] - 2026-06-17

### Added

- 新增节点模板能力，可将框选节点和连线保存为模板并再次插入。
- 新增卡片四角缩放控制。

### Changed

- 优化图片、文本、灵感、表格等节点在缩放后的内容布局。
- 优化底部工具栏展开区域和模板入口。

## [0.1.9] - 2026-06-17

### Changed

- 完成项目目录结构整理，将画布、编辑器、快捷键和共享模块迁移到 `features` / `shared` / `app` 结构。
- 移动画布节点和画布专属组件到 `src/features/canvas`。
- 移动 Tiptap 编辑器相关组件、hooks、extensions 和 utils 到 `src/features/script-editor`。
- 预留 `src/features/collaboration` 目录，用于未来协作能力扩展。

### Added

- 新增颜色选择组件和调色能力。
- 新增快捷键模块入口和设置面板模块化结构。

## [0.1.8.3] - 2026-06-17

### Changed

- 拆分 `TiptapEditor`，将工具栏、查找替换、目录、Markdown 导入导出、段落切片和状态栏迁移到独立组件与 hooks。
- 将搜索高亮扩展、文本统计和 Markdown 转换逻辑拆分到独立模块。

## [0.1.8.2] - 2026-06-17

### Changed

- 拆分 `FlowCanvas`，将画布渲染、浮层、节点命令、连线命令、剪贴板、媒体库、右键菜单和自动布局等职责迁移到独立模块。
- 新增 `CanvasViewport`、`CanvasOverlays` 和多个 canvas hooks / utils，降低主画布入口复杂度。

## [0.1.8] - 2026-06-17

### Added

- 新增快捷键设置面板。
- 新增画布快捷键配置能力。

### Changed

- 优化底部工具栏、提示气泡、属性面板和多个节点组件的交互细节。
- 调整时间轴节点和编辑器部分 UI 表现。

## [0.1.7] - 2026-06-17

### Changed
